得心应手使用Git

作者: 丰雪 分类: git, 随手笔记 发布于: 2019-03-20 16:04 浏览(2,287) 没有评论

Git------强大的分布式版本控制系统

首先需要安装git然后创建本地仓库（repository）这个目录里的文件都被git管理起来

创建Git仓库只需在当前目录下初始化它 可以用ls -ah 查看本地出现了 .git文件


$ git init


把文件添加到本地仓库

$ git add filename


可以使用add添加多个文件 现在只是把文件添加到暂存区 要把文件真正存储到仓库还需要下步 git add . 所有更改的都添加到仓库 把文件提交到本地仓库



$ git commit -m "本次提交说明"

本地仓库就配置的差不多了，接下来配置远程仓库

添加远程仓库

在GitHub上 点击New repository 创建新的仓库 或者找到已经存在的库 点击clone or download 复制这个库的地址


$ git remote add origin +自己库的地址


这个样子本地仓库和远程仓库就联系在一起了

把本地库的内容发布到远程库上

首先需要克隆下远程仓库到本地

git clone +自己远程仓库地址

$ git push -u origin masterbr

这个样子git就可以使用了，接下来的是实际开发中常用的命令

查看当前状态

git status

打印日志 每次修改的内容

git log

git reflog 详细日志

撤销修改

git checkout --filename 撤销工作区的修改

git reset HEAD 撤销暂存区的修改 （add过后的）

版本回退

git reset --hard HEAD^ 回退到上一版本

git reset --hard commit_id 回退到指定版本

git reflog 查看详细日志 (最前面的编码就是commit_id)

分支管理

$ git branch dev 创建名叫dev的分支

$ git branch 显示所有分支

$ git checkout dev 切换进dev分支

$ git merge dev 合并dev分支到当前分支

$ git branch -d dev 删除dev分支

$ git diff 工作区与缓存区的差异

$ git -cached 缓存区与本地仓库的差异

$ git diff HEAD 工作区与本地仓库的区别
